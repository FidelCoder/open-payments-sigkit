# @open-payments-devkit/fixtures

Deterministic requests, Ed25519 key material, signed vectors, and verification-case matrices for the Open Payments HTTP Signatures devkit.

Included fixture groups:

- request payloads
- public and private JWK fixtures
- JWKS fixture
- signed golden vectors
- verification conformance cases

These fixtures are intended for tests, examples, and interoperability debugging across the TypeScript and Python implementations.
